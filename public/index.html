What is Ralph?

Ralph is a bash loop that turns Claude Code into an autonomous coding agent. You write a PRD with user stories, start the loop, and wake up to shipped features.

How It Works

The Ralph loop runs continuously until all your user stories are complete.

1

Write Your PRD

Define user stories in prd.json with acceptance criteria. Each story is a discrete piece of work Claude can complete.

2

Start the Loop

Run ./ralph.sh and go to sleep. Ralph spawns Claude Code with --dangerously-skip-permissions for full autonomy.

3

Claude Implements

Claude reads the PRD, picks the first incomplete story, implements it following the acceptance criteria, then commits.

4

Mark Complete

Claude updates prd.json to mark the story as done, logs progress, and the loop restarts with the next story.

5

Wake Up to Code

Morning arrives. Your features are implemented, tested, and committed. Review the git log and ship to prod.

Quick Start

Get Ralph running in under 2 minutes.

1

Download and extract

# Download the starter kit
curl -LO https://ralphmode.com/ralph-starter.zip
unzip ralph-starter.zip
cd ralph-starter
2

Edit your PRD

# Open prd.json and add your user stories
{
  "stories": [
    {
      "id": "US-1",
      "title": "Add login page",
      "passes": false
    }
  ]
}
3

Run Ralph

# Start the autonomous loop
./ralph.sh

# Go to sleep. Wake up to shipped code.
